Welcome to Our AI Agent Project

This project features an AI agent programmed in Python that utilizes the llama-index RAG (Retrieval-Augmented Generation) model. The agent is equipped with various libraries and tools necessary for its operation, including data sources and other dependencies. Additionally, the project integrates the ChatGPT API key for enhanced functionality.

Version 1 of the project is designed to interact specifically with data sources, allowing users to ask questions related to the provided data.
